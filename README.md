# ShanRakshak 

**ShanRakshak** is a civilian emergency response platform that instantly notifies nearby trained volunteers, pre-alerts the nearest hospital, and provides immediate action with first-aid guidance — closing the gap between an emergency and the first trained responder.

## Try It Live
🌐 **[Open ShanRakshak](https://shan-rakshak.vercel.app)**
No setup needed — runs directly in your browser.

## The Problem

In a medical emergency, the first few minutes are critical. Most people around an incident are untrained and helpless. Ambulances also reaches late due to traffic and other reasons. ShanRakshak bridges this gap by activating a network of nearby trained volunteers the moment an alert is raised.

## What ShanRakshak Does

- **One-tap SOS** — citizen raises an alert with their location
- **Volunteer notification** — nearby trained volunteers are alerted instantly
- **Hospital pre-alert** — nearest hospital receives advance notice
- **Live first-aid guidance** — real-time instructions for bystanders while help is on the way
- **Coordination dashboard** — tracks response status from alert to resolution
