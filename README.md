# Shanrakshak 🚑

Shanrakshak is a civilian emergency coordination platform designed to reduce response time during critical situations. It allows users to report emergencies, alerts nearby responders, and provides real-time first-aid guidance.

## 🚀 Features
- Emergency reporting system
- Real-time location detection
- Simulated volunteer & hospital alerts
- Instant first-aid guidance
l

## 🌍 Vision
To reduce the time gap between emergency occurrence and first response, potentially saving lives.
